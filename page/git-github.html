Comandos

Iniciar um repositório:

git init
git clone https://github.com/euchico/<REPOSITÓRIO>.git

Verificar o estado do repositório:

git status

Enviar o arquivo para "staging area":

git add . git add -u .

Gravar os aquivos no histórico:

git commit -am "<MENSAGEM>" git commit -m "<MENSAGEM>"

Enviar ou receber arquivos do repositório remoto:

git pull origin master git push origin master

Interações com branch:

git branch git checkout -b <NOME DO BRANCH>
